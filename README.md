# animate-shaking-bell-iOSApp

## Описание:
Игра "Дрожащий колокольчик". На экране имеется 3 слайдера для регулировки параметров: продолжительность, угол и смещение, а также кнопка для сброса настроек.

## Стэк:
- UIKit
- UIStackView
- UISlider
- UILabel
- UIButton
- UIImageView
- Custom View with xib
